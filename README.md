Instruction
The following files are the demo files from the presentation
- groverdemo.py
- ecdhHandshake.py
- mlkemHandshake.py
- bb84.py

For the demos make sure to keep pressing anykey until the end of the demo. 
Additionally, for ecdhHandshake.py, mlkemHandshake.py, and bb84.py make sure that logging is set 
to logging.DEBUG otherwise the logs will not show.

The following files have simple initial implementations of Grover's and ML-KEM
- grovers.py
- testMLKEM.py

This code was written for the final project for Spring 2026 Network Security course at CCNY.
